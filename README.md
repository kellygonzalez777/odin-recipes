# odin-recipes
I am creating a basic recipe html website using [The Odin Project](https://www.theodinproject.com/lessons/foundations-recipes) cirruculum. 
I will be using basic HTML syntax such:
- using headers `h1`, `h2`, and so forth
- using a absolute and relative links
    - an external website
    - linking to another html page
- images
- unordered list
- ordered list
- html5 boilerplate

I will be also learning how to use GitHub with:
- making good commits
- publish web projects from GitHub repository
