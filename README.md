# Odin Recipes
This project is part of the Foundation curriculum from [The Odin Project](https://www.theodinproject.com/).

The goal of this project is to build a basic recipe website using onlty HTML. Through this project, I am practicing fundamental web development concepts and becoming familiar with Git and GitHub workflows.

## Skills Demonstrated
- Creating HTML pages using the HTML5 boilerplate
- Using headings, paragraphs, and lists
- Adding images to web pages
- Creating absolute and relative links
- Organizing a multi-page website
- Writing meaningful Git commits
- Publishing a project with GitHub
